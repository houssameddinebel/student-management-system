# 🎓 Student Management System (Tkinter - MVC)

A desktop application built with Python and Tkinter to manage student data using the MVC (Model-View-Controller) architecture.

---

## 🚀 Features
- Add new students
- Update student information
- Delete students
- Display students in a table (Treeview)
- Select and edit students easily
- Show statistics:
  - Total students
  - Number of males
  - Number of females

---

## 🛠️ Technologies Used
- Python
- Tkinter (GUI)
- ttk (Treeview, Combobox)
- Object-Oriented Programming (OOP)
- MVC Architecture
